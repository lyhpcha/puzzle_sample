一個以 HTML5 + jQuery Mobile 完成的簡單遊戲，圖片分為9個圖塊，打亂圖形後拿掉第9個圖塊，
以滑塊的方式將圖塊的順序排好即可過關，每個圖塊右上角有位置提示，降低難度。
共有15張圖片，都是在台灣旅行時所拍的照片（非專業攝影，請見諒），會隨機出現不同的圖片。
操作方式很簡單，按下「遊戲開始」就可以開始玩了，有計時與步數，可以知道此次花了多少時間。
這次的APK有加上AdMob的廣告，是以plugin的方式來掛載廣告。
